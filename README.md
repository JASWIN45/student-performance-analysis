# Student Performance Analysis System

A beginner Data Science project using Python and NumPy.

## Features
- Student marks analysis
- Multiple subject support
- Grade calculation
- Pass/Fail detection
- Class topper detection
- Subject toppers
- Graph visualization
- Colored terminal output

## Technologies Used
- Python
- NumPy
- Matplotlib
- Colorama

## Graphs Generated
- Student Total Marks Graph
- Subject Average Marks Graph
